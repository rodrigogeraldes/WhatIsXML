Quando surgiu o XML?

A história da XML teve início nos anos 70 como, foi provida de uma outra linguagem elaborada pela IBM, a GML (General Markup Language), sendo esta implantada de modo a suprir a necessidade do armazenamento de dados em um formato de texto simples.

A partir da criação da GML, a IBM conseguiu fazer a classificação e processamento de todos os documentosos de maneira rápida. Em 1986, a ISO1 (organização internacional para padronização), trabalhou na padronização desta linguagem. Neste instante, foi criada a SGML (Standard Generalized Markup Language), que era a mesma GML, porém, padronizada. Sendo a SMGL uma linguagem eficaz, bastante adaptável.


Em 1989, foi criado pelo britânico Tim Berners-Lee, o HTML (Hypertext Markup Language). Linguagem de marcação criada para a construção de páginas em um site. A HTML foi derivada da SGML. Utiliza-se um conjunto de tags que determinarão a função de cada elemento.


Já em 1994, uma entidade chamada W3C (world wide web consortium(consórcio da www)), foi implantada para colocar “ordem” no HTML, formalizando suas regras, para a criação de um padrão. Mesmo sendo bem estruturado, o HTML não cumpriu tudo o que submetia e planejava para a Internet, e crescendo de uma maneira descontrolada e desordenada, o HTML se espalhou rapidamente, porém, ele não consegue suprir tudo que as aplicações necessitam. Surge então, em 1996, a XML.

Quando a XML foi fundada, pensou-se que o propósito da XML seria de substituição da HTML, porém, este pensamento logo mudou, visto que a XML foi proposta com o objetivo de suprir as limitações da HTML, complementando-as.


O XML pretendia sanar as carências do desta linguagem de marcação, no que se refere principalmente ao tratamento da informação. O HTML pode gerar uma certa confusão em quem está analisando o código, no que se refere a localização de informações, sendo esta sempre misturada entre etiquetas.Isto acaba causando uma má gestão das informações, de maneira que o XML a soluciona.
